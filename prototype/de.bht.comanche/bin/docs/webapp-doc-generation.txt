## Generate API Docs with YUIDoc ##

# inside the folder "de.bht.comanche/src/main/webapp" simply type:

yuidoc .

# the outdir is "...webapp/apidocs"
# further configurations can be done inside yuidoc.json

# check out     http://yui.github.io/yuidoc/
# especially    http://yui.github.io/yuidoc/syntax/index.html
